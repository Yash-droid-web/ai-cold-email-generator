## 🧠 What It Does

Fill in a few details about the recipient and yourself, pick a tone and length, and the app writes a fully personalised cold email for you — with a subject line and a clear call-to-action.

**Use cases:**
- 💼 Job applications & freelance outreach
- 🤝 Partnership & collaboration requests
- 📣 Sales & business development
- 🎤 Podcast / media pitches

---

## ✨ Features

- 🎯 **Personalised output** — tailored to the recipient's role and company
- 🎨 **4 tone options** — Professional, Conversational, Bold, or Warm
- 📏 **3 length options** — Short, Medium, or Detailed
- ⬇️ **Download emails** as `.txt` files
- ⚡ **Instant generation** via Claude API

## 📦 Installation

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/ai-cold-email-generator.git
cd ai-cold-email-generator
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Set your API key
```bash
export ANTHROPIC_API_KEY=your_api_key_here
```
> Get your free API key at [console.anthropic.com](https://console.anthropic.com)

### 4. Run the app
```bash
streamlit run app.py
```

---

## 🌐 Deploy for Free (Streamlit Cloud)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your repo
4. Add `ANTHROPIC_API_KEY` under **Secrets**
5. Click **Deploy** — done ✅
